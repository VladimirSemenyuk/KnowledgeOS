# Engineering Managers Folder Structure

Each folder is a Engineering Manager:

- Андрей
- Игорь
- Роман
- Юрий

Each Engineering Manager Folder contains subfolders:

- Essays
- Meetings

# Naming

Essay naming templates is:
YYYY-MM-essay.md

# Writing Style

For Russian essays, summaries, and meeting materials:
- write in Russian by default;
- use the canonical language policy and glossary from the root `AGENTS.md`;
- keep English only for role titles, company terminology, metrics, system names, direct quotes, or technical terms without a good Russian equivalent.
