# Slide Build Map

Координаты ниже даны для `1920 × 1080`.
Если Figma Slides работает в другом внутреннем масштабе, сохранять пропорции.

Общий шаблон:

- фон слайда: `#121212`;
- label: `x=128`, `y=96`;
- номер слайда: `x≈1710`, `y=92`;
- нижняя красная линия: `x=128`, `y≈995`, `w≈420`, `h=8`;
- основной заголовок: левый крупный блок;
- красный акцент: один сильный элемент на слайд.
