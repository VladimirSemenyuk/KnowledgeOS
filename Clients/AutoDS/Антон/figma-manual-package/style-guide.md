# Style Guide

## Формат

- Формат: `16:9`.
- Рекомендуемый размер в Figma Slides: `1920 × 1080`.
- Рабочая зона: примерно `128 px` слева / справа, `96 px` сверху, `85 px` снизу.
- Основная композиция: крупный левый тезис + акцентный элемент / данные справа.

## Палитра

Основные цвета:

- Background: `#121212`
- Surface: `#242424`
- Surface secondary: `#303030`
- Text primary: `#F5F5F5`
- Text muted: `#A8A8A8`
- Text dim: `#666666`
- Accent: `#E94957`
- Accent dark: `#82232D`
- White: `#FFFFFF`

Правило:
темный фон доминирует, красный акцент используется точечно: линия, крупное число, поле, полукруг, вертикальная плашка.

## Типографика

Если доступен шрифт из референса — использовать его.
Если нет, использовать:

- `Inter` или `Arial`;
- заголовки: Bold / Semi Bold;
- body: Regular.

Размеры для Figma Slides `1920 × 1080`:

- Eyebrow / label: `28 px`, Semi Bold, uppercase, `#E94957`.
- Slide number: `24 px`, Semi Bold, `#666666`.
- Главный заголовок: `90–120 px`, Bold.
- Вторичный крупный тезис: `60–78 px`, Bold / Semi Bold.
- Body / пояснение: `34–44 px`, Regular.
- Bullet list: `32–36 px`, Regular.
- Подписи к метрикам: `28–34 px`, Regular, `#A8A8A8`.

Line height:

- Заголовки: `1.05–1.12`.
- Body: `1.25–1.4`.
- Bullet list: `1.35–1.45`.

## Повторяющиеся элементы

На каждом слайде:

- label в левом верхнем углу;
- номер слайда в правом верхнем углу;
- короткая красная линия внизу слева;
- минимум один визуальный элемент: красное поле, круг, линия, блок, метрика.

## Shape Language

Использовать:

- крупные красные круги / полукруги с bleed за границы слайда;
- красные вертикальные акцентные полосы;
- темные прямоугольники без скругления;
- длинные горизонтальные линии;
- асимметричную композицию.

Не использовать:

- много карточек на одном слайде;
- декоративные градиенты;
- иконки ради иконок;
- скругленные UI-плашки;
- плотные таблицы;
- одинаковый layout на каждом слайде.
