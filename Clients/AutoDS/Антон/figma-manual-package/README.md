# Ручной Figma-пакет: AI-native PDLC

Пакет для ручной сборки слайдов в Figma Slides по материалу:

- [[Clients/AutoDS/Антон/ai-transformation-mini-deck|ai-transformation-mini-deck]]

Цель:
быстро собрать 12 слайдов в стиле референсной Figma-презентации без Figma MCP.

## Состав пакета

- `style-guide.md` — палитра, типографика, сетка, общие правила.
- `slide-build-map.md` — покадровая инструкция: текст, композиция, элементы.
- `slide-copy.md` — чистый текст для копирования в Figma.
- `previews/` — PNG-превью каждого слайда и общий `montage.webp`.

## Как использовать

1. Открыть Figma Slides.
2. Создать новую секцию: `AI-native PDLC`.
3. Создать 12 слайдов формата `16:9`.
4. Взять стиль из `style-guide.md`.
5. По каждому слайду идти по `slide-build-map.md`.
6. Сверяться с `previews/slide-XX.png`.

## Базовая логика

Это не документальная презентация.

Слайды должны работать как executive narrative:

- один главный тезис на слайд;
- минимум текста;
- крупная типографика;
- темная сцена;
- красный акцент как навигационный и смысловой маркер;
- без карточной перегрузки и без generic consulting language.
